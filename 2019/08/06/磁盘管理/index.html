磁盘管理

发表于 2019-08-06 阅读次数：

df 查看磁盘分区使用状况

  • -l 仅显示本地磁盘（默认）

  • -a 显示所有文件系统的磁盘使用状况

  • -h 以1024进制显示磁盘容量

  • -H 以1000进制显示磁盘容量

  • -T 显示磁盘分区类型

  • -t 显示指定类型文件系统的磁盘分区

  • -x 不显示指定类型文件系统的磁盘分区

du 统计磁盘上的文件大小

  • -b 以byte为单位统计文件

  • -k 以KB为单位统计文件

  • -m 以MB为单位统计文件

  • -h 以1024进制显示磁盘容量

  • -H 以1000进制显示磁盘容量

  • -s 指定统计目标

MBR分区

  • fdisk -l 列出当前分区表

  • fdisk 路径 进入分区模式

  • m 查看命令帮助信息

  • n 增加新分区

GPT分区

  • parted 启动分区工具

  • help 帮助信息

  • select 切换目标磁盘

  • mklabel gpt 指定分区类型 [ mklabel msdos 指定MBR分区 ]

  • print 查看当前硬盘分区详情

  • printall 查看所有硬盘分区详情

  • mkpart 增加新分区

  • rm 分区编号 删除分区

  • quit 退出分区工具
